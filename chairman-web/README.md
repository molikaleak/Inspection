# Inspector Chairman Intelligence Dashboard

A high-fidelity tactical dashboard for executive field oversight, built with React, Vite, and Tailwind CSS.

## 🚀 Quick Start
```bash
npm install
npm run dev
```

## 🛠 UX Architecture & Tactical Flow

### The Operational Loop
1.  **Dashboard (Nerve Center)**: Scan real-time compliance vectors and recent report streams.
2.  **Filter (Tactical Drawer)**: Isolate data by Project, Inspector, or Status using a **Zero-Click** grid interface.
3.  **Dossier (Deep-Dive)**: Review evidence-backed inspection reports with foreman merit scores and photo logs.
4.  **Protocol (Export)**: Generate print-optimized executive operational reviews.

For a detailed breakdown of the interaction design, see [UX_PROTOCOL.md](./UX_PROTOCOL.md).

## 💡 Room for Improvement (Future Roadmap)

While the current system provides high-density intelligence, several "Future Vectors" have been identified for development:

### 1. Predictive Compliance Analysis
Integrate machine learning to flag "At-Risk" projects based on historical foreman scores and inspector trends *before* a safety violation occurs.

### 2. Live Geospatial Tracking
Implement a map-based view to visualize inspector travel paths and real-time project distribution across the region.

### 3. Direct Operational Nudges
Enable the Chairman to send "Tactical Nudges" (pings) directly to field inspectors' tablets from the dashboard report view to address critical compliance gaps instantly.

### 4. Comparison Modeling
Allow side-by-side comparison of Safety Execution metrics between different projects or quarters to identify systemic operational patterns.

## 🎨 Design Philosophy: "The Obsidian Forest"
*   **High-End Minimalism**: Dark-mode aesthetic that reduces cognitive load by prioritizing data over chrome.
*   **Information Density**: Engineered for executive users who need to process large volumes of operational data quickly.
*   **Tactical Colors**: Contextual color-coding (Emerald/Amber/Cyan) for immediate status recognition.
